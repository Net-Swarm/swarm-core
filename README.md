### // THE NEURAL CENTER
`swarm-core` is the high-performance, headless C++ backbone of the **Net-Swarm** ecosystem. It is designed to handle systemic emergence, particle math, and tactical routing with zero garbage collection overhead.

### // CAPABILITIES
- **Zero-GC Architecture:** Manual memory management via RAII for stutter-free simulation.
- **Dual-Target Execution:** Compiled for native DX12 environments and WebAssembly (Wasm) via Emscripten.
- **Agnostic Logic:** Pure physics and telemetry—detached from UI and rendering shells.

### // THE ECOSYSTEM
This core feeds two primary nodes:
1. **[swarm-engine]**: The native desktop environment (DirectX 12 / IMGUI).
2. **[net-swarm]**: The centralized web destination (Next.js / Wasm).
